# CryptoWeather Nexus

## Description
A modern dashboard combining live crypto prices, weather data, and crypto news with WebSocket notifications.

## Tech Stack
- Next.js 13+
- Redux Toolkit
- React
- Tailwind CSS
- WebSocket

## Features
- Real-time weather for 3 cities
- Crypto-related news
- Notification toasts
- Detail pages for each city and crypto
- Responsive design

## Setup
```bash
npm install
npm run dev
